**🏆 DIU LeaderBoard – Project Overview**
**📱 Platform:**
A mobile app developed for students of Daffodil International University (DIU).
Built using Flutter for fast, cross-platform performance.

**🔑 Core Features**
**🔐 Student Login**
<img src="https://github.com/user-attachments/assets/698ca5ad-ed05-411f-bd57-750e4c0d55ae" height="500" width="200"/>
Each student logs in with their credentials to access their academic data securely.

**📊 Ranking System**
<img src="https://github.com/user-attachments/assets/2a03c54d-ad2b-4e97-bea6-f76d2e886c4b" height="500" width="200"/>
Displays the student’s rank within their batch, helping them track their academic standing among peers.

**📚 Semester-wise Result**
<img src="https://github.com/user-attachments/assets/75e5f147-57c1-4e9b-adb5-ed708a3f28ee" height="500" width="200"/>

<img src="https://github.com/user-attachments/assets/409a7220-0f00-438a-b74d-73e2f5dd4737" height="500" width="200"/>

**💡 Retake Analysis + Subject Suggestions**
